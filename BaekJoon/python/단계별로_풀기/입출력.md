### **10869번 사칙연산**

```py

print(a+b)
print(a-b)
print(a*b)
print(int(a/b))
print(a % b)
```

### **10430 나머지**

```py
a, b, c = map(int, input().split())

print((a + b) % c)
print(((a % c) + (b % c)) % c)
print((a * b) % c)
print(((a % c) * (b % c)) % c)
```

### **2588 곱셈**

```py
a = int(input())
b = input()

print(a * int(b[2]))
print(a * int(b[1]))
print(a * int(b[0]))
print(a * int(b))
```
